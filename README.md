# General info

The application ensures that the user stays socially connected during busy days with Friends, Family, and Colleagues. It is an easy to use andoid application that allows users to add Inputs (the suggested "sparks") and the application will generate Outputs, these outputs are text messeges that can be sent to Friends, Family or Colleagues. The user's Inputs / Sparks represents times of the day : Morning, Mid-morning, Afternoon, Afternoon Snack, Dinner, Night.
